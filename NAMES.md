## Selection Criteria

1. **npm available** - Required for publishing
2. **Domain availability** - At least .dev or .io preferred
3. **Memorable** - Easy to say and remember
4. **Descriptive** - Conveys the purpose
5. **Unique** - Stands out from existing tools

## Recommendations

For "reimagining GitHub for human-AI collaboration":

1. **githuman** - Positions as the human counterpart to GitHub in AI age
2. **pairgit** - Evokes pair programming, human-AI style
3. **aidiff** - Simple, clear, technical
4. **stageguard** - Protective, guarding staged changes
5. **agentveto** - Strong message about human control
